\hypertarget{index_sec_intro}{}\doxysection{Introduction}\label{index_sec_intro}
\begin{DoxyParagraph}{}
The {\itshape Nimble Game Engine Library} (or {\itshape Nimble Library}) is the \char`\"{}engine\char`\"{} in {\itshape Nimble Game Engine}. Developers can use the library to make their own games, and it is recommended they use the {\itshape Nimble Game Engine Application} to do so.
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
As well as having useful tools to make low-\/level game development easier, the library comes with its own error and crash handler that runs by default in the background of each game, should the developer choose to call the engine initialization function.
\end{DoxyParagraph}
