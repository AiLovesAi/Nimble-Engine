NIMBLE_EXPORT int32_t nEngineInit ( void  )

Initialized the game engine.

Initialized the game engine by starting the error/crash handlers, logger, and thread manager.

Example:
#include <stdint.h>
#include <Nimble/NimbleEngine.h>
int main(int argc, char ** argv)
{
if (nEngineInit() == NERROR)
{
fprintf(stderr, "Failed to initialize Nimble.");
exit(EXIT_FAILURE);
}
printf("Successfully initialized game engine.");
exit(EXIT_SUCCESS);
}
Returns
NSUCCESS is returned if successful; otherwise NERROR is returned and a corresponding error is sent to the error callback set by nErrorHandlerSetErrorCallback().
