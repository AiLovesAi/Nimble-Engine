const int32_t  error,

)

Describes an error and returns a string.

Example:

#include <stdio.h>
#include <stdlib.h>
#include <Nimble/NimbleEngine.h>
int main(int argc, char ** argv)
{
char * errorString;
nErrorToString(errorString, NERROR_FILE_NOT_FOUND, "example.txt");
if (errorString == NULL)
{
fprintf(stderr, "Failed to get error string.\n");
exit(EXIT_FAILURE);
}
printf("NERROR_NULL as string: %s\n", errorString);
return EXIT_SUCCESS;
}
Parameters
[out]dstThe destination to store the string describing error.
[in]errorThe error to get described.
[in]infoRelevant information, such as a file location, that could help diagnose the error. This can be NULL.
Returns
dst is returned if successful; otherwise NULL is returned.
