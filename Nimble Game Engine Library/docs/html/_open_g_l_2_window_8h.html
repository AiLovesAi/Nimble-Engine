NIMBLE_EXPORT int32_t nWindowCreate ( const char *  title,
const uint8_t *  iconData,
const uint32_t *  dimensions,
const int32_t  flags 
)

Creates a window.

Creates a window with dimensions dimensions, title title, icon iconData, and flags flags.

Example:
uint8_t *iconData = nWindowLoadIcon("icon.ico");
uint32_t *dimensions = nWindowDimensions(NWINDOW_MONITOR_DEFAULT, windowPosX,
windowPosY, windowWidth, windowHeight);
int32_t window = nWindowCreate("Hello World", iconData, dimensions,
NWINDOW_FULLSCREEN | NWINDOW_CURSOR_HIDDEN);
Parameters
[in]titleThe title that is displayed at the top of the window.
[in]iconDataThe image data of the icon for the window.
[in]dimensionsThe position and size of the window, generated by nWindowDimensions().
[in]flagsThe flags set when creating the window, defined by nWindowFlags_t.
Returns
The window ID generated by the new window is returned if successful, otherwise NERROR is returned.
Note
If an error occurs, a description of the error will be sent to the error handler .
