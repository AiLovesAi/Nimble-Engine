Nimble Game Engine Library

Introduction

The Nimble Game Engine Library (or Nimble Library) is the "engine" in Nimble Game Engine. Developers can use the library to make their own games, and it is recommended they use the Nimble Game Engine Application to do so.
As well as having useful tools to make low-level game development easier, the library comes with its own error and crash handler that runs by default in the background of each game, should the developer choose to call the engine initialization function.
