◆ nFree()

NIMBLE_INLINE void* nFree ( void *  ptr)

Frees a pointer.

Frees a pointer and returns NULL to allow the invoker to optionally nullify the pointer. Should the freed pointer location be needed again, no assignment is necessary.

Example:

#include <stdio.h>
#include <stdlib.h>
#include <Nimble/NimbleEngine.h>
int main(int argc, char **argv)
{
char original[] = "Hello world.";
char *new = nAlloc(sizeof(original) + 1);
new[sizeof(original) - 1] = '\n';
new[sizeof(original)] = '\0';
puts(new);
new = nFree(new);
if (new)
{
puts("We will get here only if we use nFree(new) with no assignment!");
}
return EXIT_SUCCESS;
}
Parameters
[in]ptrThe pointer to free.
Returns
NULL is always returned.

NIMBLE_EXTERN NIMBLE_FREEME size_t nStringCopy ( char *restrict  dst,
const char *restrict  src,
const size_t  len 
)

Copies len characters from src to dst.

Copies len characters from src to dst. The string is always null terminated. If there is already a null terminator, no more characters are copied.

Example:

#include <stdio.h>
#include <stdlib.h>
#include <Nimble/NimbleEngine.h>
int main(int argc, char **argv)
{
char original[] = "Hello world.";
char *new = nAlloc(sizeof(original));
nStringCopy(original, new, NCONST_STR_LEN(original));
printf("New string: %s\n", new);
return EXIT_SUCCESS;
}
Parameters
[out]dstThe pointer to the destination.
[in]srcThe pointer to free.
[in]lenThe number of characters to copy. This must not include the null terminator.
Returns
The number of successfully copied bytes is returned.
