void(*)(const nErrorInfo_t errorInfo)  errorCallback,
void(*)(const nErrorInfo_t errorInfo)  crashCallback 
)

Initialized the game engine.

Initializes the game engine by setting up the error/crash handlers, logger, and thread manager.

Parameters
[in]argsThe arguments past to main (usually argv).
[in]argcThe argument count of args.
[in]errorCallbackThe error callback function that is called when an error occurs. This can be NULL to use the default callback.
[in]crashCallbackThe crash callback function that is called when the engine crashes. This can be NULL to use the default callback.
Returns
NSUCCESS is returned if successful; otherwise an error is returned and a corresponding error is sent to the error callback set by nErrorHandlerSetErrorCallback().
