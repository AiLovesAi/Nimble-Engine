◆ nThreadCreate()

int nThreadCreate ( nThread_t *  thread,
nThreadRoutine_t(*)(void *)  start,
void *  data 
)

Creates a thread.

Creates a thread starting at start() where data is passed, whose identity is stored in thread with attributes attributes.

Parameters
[out]threadThe thread identity of the created thread.
[in]startThe start function for the thread to start in. This function should take a void * argument, which data is sent to, and should return its return value as a void *.
[in]dataA pointer to the argument to pass to start.
Returns
NSUCCESS is returned if successful; otherwise an error is returned and a corresponding error is sent to the error callback set by nErrorHandlerSetErrorCallback().
Note
If a thread is not later joined to check its result, call nThreadDetach(). To be fully portable, threads must return an integer value.
