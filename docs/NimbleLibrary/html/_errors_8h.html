◆ nErrorGetStacktrace()

NIMBLE_EXTERN NIMBLE_FREEME char* nErrorGetStacktrace ( char *  dst,
size_t *  stackLen,
size_t *  stackLevels 
)

Returns the current stack trace as a string.

Returns the current stack trace as a string, and sets the stackLen of the string and levels of the stack.

Example:

#include <stdio.h>
#include <stdlib.h>
#include <Nimble/NimbleEngine.h>
int main(int argc, char **argv)
{
size_t stackLen, stackLevels;
char *stack;
nErrorGetStacktrace(stack, &stackLen, &stackLevels);
if (stack == NULL)
{
fprintf(stderr, "Failed to get stack trace.\n");
exit(EXIT_FAILURE);
}
printf("Last %d levels of stack trace: %s\n", stackLevels, stack);
return EXIT_SUCCESS;
}
Parameters
[out]dstThe destination to store the stacktrace string. This can be NULL.
[out]stackLenThe length of the string returned. This can be NULL.
[in,out]stackLevelsThe number of levels found on the stack. If the value of this is nonzero when passed, it will act as the maximum number of stack levels to be returned in dst. This can be NULL.
Returns
dst is returned if successful; otherwise the program crashes.
Note
Each time a function is called, it is added to the stack. When a function returns, it is removed from the stack.

NIMBLE_EXTERN nint_t nErrorSetCallback ( void(*)(const nint_t error, const time_t errorTime, const char *errorDesc, const size_t errorDescLen, const char *stack, const size_t stackLen)  callback)

Sets the callback function to handle errors.

Sets the callback function callback that gets called when an error occurs.

Example:
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
#include <Nimble/NimbleEngine.h>
void errorHandler(const nint_t error, const char * errorDesc,
const size_t errorDescLen, const time_t errorTime, const char * stack,
const size_t stackLen)
{
struct tm *timeInfo = localtime(&errorTime);
const char format[] = "%x %X %Z";
const char example = "01/01/2020 16:30:45 GMT"
char *timeString = nAlloc(sizeof(example));
if (timeString == NULL)
{
fprintf(stderr, "Failed to allocate to timeString.\n");
return;
}
strftime(timeString, sizeof(example), format, timeInfo);
fprintf(stderr, "\nAn error occurred at %s:\nError description: "\
"%s\nStack trace: %s\n\n", timeString, errorDesc, stack);
}
int main(int argc, char **argv)
{
if (nErrorHandlerSetErrorCallback(errorHandler) != NSUCCESS)
{
fprintf(stderr, "Could not set error callback for Nimble.\n");
return EXIT_FAILURE;
}
printf("Successfully set error callback for Nimble.\n");
return EXIT_SUCCESS;
}
Parameters
[in]callbackThe function that gets called when an error occurs.
Returns
NSUCCESS is returned if successful; otherwise NERROR is returned.
Note
Check nErrorHandlerDefault() for parameter information.

Sends an error to the error callback.

Sends an error to the error callback defined by nErrorHandlerSetErrorCallback(), and determines whether or not crashing is necessary.

Example:

#include <stdio.h>
#include <stdlib.h>
#include <Nimble/NimbleEngine.h>
int main(int argc, char **argv)
{
char exampleFilePath[] = "example.txt";
if (nErrorThrow(NERROR_FILE_NOT_FOUND, exampleFilePath,
sizeof(exampleFilePath)) != NSUCCESS)
{
fprintf(stderr, "Failed to throw error.\n");
exit(EXIT_FAILURE);
}
printf("Successfully threw error.\n");
return EXIT_SUCCESS;
}
Parameters

[in]infoLenThe length of the info argument. A length of zero (0) uses strlen() to determine length. diagnose the error. This can be NULL.
Note
